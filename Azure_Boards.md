**what is Azure Boards**
  Azure boards is a work tracking and Project management tool inside the azure devops this will be used to plan, track and manage work

**What problems does Azure Boards solve?**

  Who is working on what?
  What is planned for this sprint?
  What is done / in progress / blocked?
  How much work is remaining?
